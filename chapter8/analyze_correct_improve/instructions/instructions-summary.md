Open the files _index.html_ and _styles.css_. The _styles.css_ file is found in the _/css_ folder.

Review the code in each file, and refer to **Figures 8-69** and **8-70** for use as a guide as you update these files.

The _index.html_ webpage contains a `form` with several input controls. Add a `label` for each input control in the `form`.

Wrap the `form` elements for the Customer Information within one `fieldset` and the `form` elements for the Payment Information in a second `fieldset`. Add an appropriate `legend` to each `fieldset`.

Create a submit button at the bottom of the `form`.

Add a placeholder value to the "Expiration Date" input control.

Add style rules to _styles.css_ for the `form`, and the elements within the `form`.

Finally, add a tablet media query with appropriate breakpoints in the file _styles.css_ and add styles for that viewport.

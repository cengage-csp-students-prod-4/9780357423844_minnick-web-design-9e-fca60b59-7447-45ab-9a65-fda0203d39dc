## Task 08
In _styles.css_, create a tablet media query with appropriate breakpoints for the tablet screen size.

Add the following rules to the tablet media query. 
* Create a style rule for the `form` element that:
    - sets the `width` to **80%**
    - sets the top and bottom `margin` to **0** (zero), and the left and right `margin` to   **auto** 
*  Create a style rule for the `grid` class that:
   - sets the `display` to a **grid**
   - sets the `grid-template-column` value to **auto auto**
   - sets the grid's `gap` value to **20px**
*  Create a style rule for the `btn` class that:
   - sets the `grid-column` value to **1 / span 2**
   - sets `padding` to **3%**.

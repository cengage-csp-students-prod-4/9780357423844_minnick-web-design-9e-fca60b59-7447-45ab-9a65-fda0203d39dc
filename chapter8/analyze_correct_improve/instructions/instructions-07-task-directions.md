## Task 07
In the file _styles.css_, add the following style rules for the form elements.
* Create a style rule for the `fieldset` and `input` elements that sets the `margin-bottom` to **2%**
* Create a style rule for the `fieldset` `legend` that:
   - sets the `font-size` to **1.25em**
   - sets the `font-weight` **bold**
* Create a style rule for the `label` that:
   - sets the `display` to **block**
   - sets `padding-top` to **3%**
* Create a style rule for the `btn` class selector that:
  - sets the `border` to **none**
  - sets the top and bottom `margin` to **0** (zero) and the left and right `margin` to **auto**
  - sets the `display` to **block**
  - sets the `padding` to **5%**
  - sets the `background-color` to **003399**
  - sets the `font-size` to **1.25em**
  - sets the `border-radius`  to   **10px**
  - sets the `color` to **white** (fff)

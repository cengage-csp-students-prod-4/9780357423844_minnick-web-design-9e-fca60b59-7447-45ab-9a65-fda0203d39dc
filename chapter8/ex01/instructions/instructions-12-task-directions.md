## Task 12

In the _styles.css_ file, within the mobile viewport style rules and above the comment for the `footer`, add the following comment:

```css
/* Style rules for form elements */
```

Add the following style rules for a mobile viewport below that comment:

- Create a style rule for the `fieldset` and `input` selectors that sets the `margin-bottom` to **2%**
- Create a style rule for the `fieldset legend` selector that sets the `font-weight` to **bold** and `font-size` to **1.25em**
- Create a style rule for the `label` selector that sets the `display` to **block** and sets the top padding to **3%**
- Create a style rule for the `form #submit` selector that:
  - sets the top and bottom margins to **0** and left and right margins to **auto**
  - sets `display` to **block**
  - sets `padding` to **3%**
  - sets `background-color` to **b7b7e1**
  - sets `font-size` to **1em**

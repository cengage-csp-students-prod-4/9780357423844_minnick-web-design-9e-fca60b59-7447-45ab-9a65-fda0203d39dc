## Task 06

In _styles.css_ find the desktop media query.

Add the following comment after the style rule for `aside`: `/* Style rules for table */`

Below the comment, add the following style rules:

- Create a style rule for the `table` selector that:
  - specifies a `border` with values of **1px solid #2a1f14**
  - sets the `border-collapse` property to `collapse`
  - sets top and bottom margins to **zero** and left and right margins to **auto**
- Create a style rule for the `caption` selector that:
  - sets a `font-size` of **1.5em**
  - sets a `font-weight` of **bold**
  - sets the `padding` to **1%**
- Create a style rule for `th` and `td` selectors that:
  - sets a `border` with values of **1px solid #2a1f14**
  - sets the `padding` to **1%**
- Create a style rule for the `th` selector that:
  - sets the `background-color` to **2a1f14**
  - sets the `color` to **white** (fff)
  - sets the `font-size` to **1.15em**
- Create a style rule for the `tr` selector that sets the `background-color` to **#deccba** for odd rows.

Open the files _index.html_ and _styles.css_. The _styles.css_ file is found in the _/css_ folder. 

Review the code in each file.

Add a `table` to _index.html_, then add a `caption` to the `table`.

Add the styles for the `table` to _styles.css_.

Add `tr`, `th` and `td` tags to the `table` to create an html table with five rows and columns. Then add the data to the cells in the table.

Add the styles for the inner elements of the `table` to _styles.css_.

## Task 01

In your _/music_ folder, create a new subfolder named **media**. Move the _media.zip_ file into your new _music/media_ folder by either drag and drop or copy and paste. You can unzip the contents by using the following terminal command:

```bash
cd ./music/media
unzip media.zip
```

> The first command moves the current working directory to the new _./music/media_ folder. The second command unzips the zip file.

## Task 01

In your _/rescue_ directory, create a new subfolder named **media**. Move the _media.zip_ file into your new _/rescue/media_ folder by either drag and drop or copy and paste. You can unzip the contents by using the following terminal command:

```bash
cd ./rescue/media
unzip media.zip
```

> The first command moves the current working directory to the new _./rescue/media_ folder. The second command unzips the zip file.
